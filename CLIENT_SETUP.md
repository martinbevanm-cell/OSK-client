OSK Frontend Client Setup

This package runs only the frontend and connects to the deployed backend.
No backend setup is required.

Before you start

1. Install Node.js LTS from https://nodejs.org/en/download
2. Restart your computer after install.

Windows (easy way)

1. Open this folder.
2. Double-click START.bat
3. Wait for the browser to open.
4. Use http://localhost:3000

macOS (easy way)

1. Open this folder.
2. Double-click START.command
3. If macOS blocks it, right-click START.command and choose Open.
4. Wait for the browser to open.
5. Use http://localhost:3000

Important

1. Keep the terminal window open while using the app.
2. To stop the app, close that terminal window.

If something fails

1. Take a screenshot of the terminal window.
2. Send the screenshot to the developer.
